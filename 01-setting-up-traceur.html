Steps

Step 1: Install nodejs

To install node js, go to the nodejs.org and download the node. To check the installed version of nodejs, write the command node -v in cmd window.

Step 2: Install treasure

Open command prompt -> write command => npm install traceur -g to instal traceur. After instalation write command traceur to check it is installed or not.

To compile the js file write the command at the js file's directory path in cmd => traceur --out outputFileName.js inputFileName.js
